## 📋 任务概述

在Novel Builder Flutter应用的搜索界面添加"通过URL添加小说"功能，用户输入小说详情页URL后，应用获取该URL对应小说的信息并展示章节列表。

### 核心需求
- 在搜索界面增加一个按钮
- 点击后弹出对话框让用户输入小说详情页URL
- 确认后展示该URL的章节列表（复用现有章节列表界面）

### 用户选择
- **API方式**: 新增一个接口，返回章节信息和章节列表
- **交互方式**: 弹出对话框(Dialog)
- **站点支持**: 所有已启用的站点

## 🔧 技术细节

### API契约
```yaml
GET /novel-by-url?url={小说URL}
Headers:
  X-API-TOKEN: {token}

Response 200:
{
  "novel": {
    "title": "小说标题",
    "author": "作者",
    "url": "原始URL",
    "cover_url": "封面URL",
    "description": "简介"
  },
  "chapters": [
    {
      "title": "第一章",
      "url": "章节URL"
    }
  ]
}

Response 400:
{
  "detail": "不支持的URL站点"
}

Response 500:
{
  "detail": "获取小说信息失败"
}
```

### UI流程
```
搜索界面
  ↓ 点击 [添加URL] 按钮
弹出URL输入对话框
  ↓ 输入URL → 点击确认
  ↓ API调用中（显示加载状态）
  ├─ 成功 → 导航到章节列表界面
  └─ 失败 → 显示错误提示，返回对话框
```
