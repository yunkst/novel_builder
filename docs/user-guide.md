# Novel Builder 用户指南

欢迎使用 Novel Builder！本文档帮助你快速上手这个现代化的全栈小说阅读平台。

## 📲 安装

### 下载应用

1. 访问 [GitHub Releases](https://github.com/yunkst/novel_builder/releases) 页面
2. 下载最新版本的 APK 文件
3. 在 Android 设备上安装（可能需要开启"允许安装未知来源应用"）

### 系统要求
- Android 5.0 (API 21) 及以上
- 建议至少 100MB 存储空间

---

## 🚀 首次启动

### 1. 配置服务器连接

首次打开应用后，需要配置后端服务地址：

1. 进入 **设置** 页面
2. 找到 **服务器配置** 选项
3. 填写后端 API 地址（例如 `http://192.168.1.100:3800`）
4. 填写 API Token（与后端 `.env` 中 `NOVEL_API_TOKEN` 一致）

> **提示**：如果你使用 Docker Compose 部署了后端，API 地址通常是 `http://<服务器IP>:3800`

### 2. 配置 AI 功能（可选）

如果你需要使用 AI 增强功能：

1. 进入 **设置** → **AI 配置**
2. 填写 **LLM API URL**（OpenAI 兼容地址，如 `https://api.deepseek.com/v1`）
3. 填写 **LLM API Key**
4. 可选填写 **默认模型**（如 `deepseek-chat`）
5. 可选填写 **AI 作家设定**（自定义 AI 写作风格）

## 📚 核心功能

### 搜索小说

1. 点击底部导航栏的 **搜索** 图标
2. 在搜索框输入小说名称或关键词
3. 选择搜索站点（默认搜索所有活跃站点）
4. 浏览搜索结果，点击小说查看详情

### 书架管理

- **添加小说**：在搜索结果或小说详情页点击"加入书架"
- **阅读进度**：应用自动记录你的阅读位置，下次打开继续阅读
- **批量管理**：长按书架中的小说可进行批量操作
- **分组管理**：支持多个书架分组

### 阅读书籍

1. 在书架中点击小说进入详情页
2. 选择要阅读的章节
3. 阅读界面支持：
   - **翻页模式**：点击屏幕左右侧翻页，或使用音量键
   - **滚动模式**：上下滑动阅读
   - **亮度调节**：在阅读界面调节亮度
   - **字体设置**：调整字体大小、行间距
   - **主题切换**：日间/夜间模式、自定义背景色

### 离线缓存

1. 在小说详情页点击 **缓存** 按钮
2. 选择要缓存的章节范围
3. 等待缓存完成（后台下载）
4. 缓存完成后即可离线阅读

## 🤖 AI 增强功能

### 场景插图

在阅读界面，可以为当前段落生成 AI 场景插图：

1. 长按段落文本
2. 选择 **生成插图** 选项
3. AI 将分析段落内容生成场景描述
4. 确认后通过 ComfyUI 生成插图（需配置 ComfyUI 服务）

### 角色卡提取

1. 在小说详情页进入 **角色管理**
2. 点击 **提取角色** 按钮
3. AI 将自动分析已缓存章节中的角色信息
4. 提取的角色包含：姓名、外貌、性格、背景等

### 人物关系图

1. 在角色管理页面点击 **关系图** 按钮
2. 查看角色之间的可视化关系网络
3. 支持拖拽、缩放交互
4. 点击节点查看角色详情

### AI 对话助手（Hermes Agent）

1. 在角色详情页点击 **聊天** 按钮
2. 与 AI 扮演的角色进行对话
3. 支持多角色群聊模式
4. AI 会根据角色设定和小说上下文进行回复

### AI 写作辅助

在阅读界面，可以使用 AI 写作辅助功能：

- **段落重写**：选中段落后选择"重写"
- **全文重写**：对整章内容进行风格转换
- **章节摘要**：自动生成章节内容摘要
- **背景摘要**：提取章节背景设定信息

## 🎯 高级功能

### TTS 朗读

在阅读界面启用文字转语音功能，支持：
- 播放/暂停控制
- 语速调节
- 后台播放

### 提纲管理

在小说详情页管理提纲：
- 创建章节提纲
- 编辑提纲内容
- 按提纲组织写作思路

### 备份与恢复

- **备份数据**：设置 → 备份 → 导出书架数据
- **恢复数据**：设置 → 备份 → 导入书架数据
- 备份包含书架、阅读进度、缓存任务信息

### 章节重排

在小说详情页可以自定义章节顺序：
- 长按章节拖拽排序
- 支持插入自定义章节

## ❓ 常见问题

### 无法连接到后端服务
- 检查手机和后端服务是否在同一网络
- 确认后端 API 地址和端口正确
- 检查 API Token 是否匹配

### 搜索无结果
- 部分站点可能暂时不可用
- 尝试更换搜索关键词
- 检查网络连接和代理设置

### AI 功能无法使用
- 确认已在 AI 配置中填写 API URL 和 Key
- 检查 LLM 服务是否正常运行
- 确认 API Key 有足够的调用额度

### 缓存失败
- 检查后端服务是否运行正常
- 检查目标站点是否可访问
- 检查网络代理配置

### 应用更新
- 应用会自动检测新版本
- 也可以在 GitHub Releases 页面手动下载最新版本
- 安装新版本不需要卸载旧版本（数据会保留）
