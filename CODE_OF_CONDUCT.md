# 贡献者行为准则

## 我们的承诺

为了营造一个开放和友好的环境，我们作为贡献者和维护者承诺，无论年龄、体型、残疾、种族、性别认同和表达、经验水平、国籍、个人形象、种族、宗教或性取向如何，参与我们项目和社区的每个人都能享受无骚扰的体验。

## 我们的标准

有助于创造积极环境的行为包括：

* 使用友好和包容的语言
* 尊重不同的观点和经验
* 优雅地接受建设性批评
* 关注对社区最有利的事情
* 对其他社区成员表示同情

不可接受的行为包括：

* 使用性化的语言或图像，以及不受欢迎的性关注或性骚扰
* 恶意评论、侮辱/贬损评论，以及个人或政治攻击
* 公开或私下骚扰
* 未经明确许可发布他人的私人信息，如物理或电子地址
* 在专业环境中可能被合理认为不适当的其他行为

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并期望对任何不可接受的行为采取适当和公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝与行为准则不符的评论、提交、代码、wiki 编辑、问题和其他贡献，或者暂时或永久禁止任何他们认为有不适当、威胁、冒犯或有害行为的贡献者。

## 适用范围

本行为准则适用于项目空间内，也适用于个人在公共空间代表项目或其社区时。代表项目或社区的示例包括使用官方项目电子邮件地址、通过官方社交媒体账户发布或在线上或线下活动中担任指定代表。项目的代表可以由项目维护者进一步定义和澄清。

## 执行

可以通过 [yedazhi@c2h4.cn](mailto:yedazhi@c2h4.cn) 联系项目团队来报告辱骂、骚扰或其他不可接受的行为。所有投诉都将被审查和调查，并将产生被认为必要和适合情况的回应。项目团队有义务对事件报告者保密。具体执行政策的更多细节可能会单独发布。

不善意遵守或执行行为准则的项目维护者可能面临项目领导层其他成员确定的临时或永久后果。

## 社区指导原则

### 🤝 友好和包容
* 欢迎新贡献者，耐心回答问题
* 使用友善的语言，避免攻击性言辞
* 尊重不同的技能水平和经验背景

### 🎯 专注和建设性
* 讨论保持专注，避免偏离主题
* 提供建设性的反馈和建议
* 接受专业分歧，寻求共识

### 🔍 尊重和尊重
* 尊重不同的观点和经验
* 承认他人的贡献和努力
* 避免做出假设或刻板印象

### 📚 持续学习
* 保持开放心态，愿意学习
* 承认错误并及时纠正
* 寻求理解而不是评判

## 具体指导

### 在 Issue 和 Pull Request 中
* 使用清晰的描述性标题
* 提供足够的上下文和重现步骤
* 保持评论相关和建设性
* 感谢他人的帮助和贡献

### 在代码审查中
* 提供具体和可操作的建议
* 解释建议背后的原因
* 认可代码中的优秀部分
* 保持耐心和专业的语调

### 在社区讨论中
* 保持讨论相关和专注
* 避免重复提出已经解决的问题
* 尊重不同的意见和方法
- 帮助引导讨论回到正轨

## 处理冲突

### 当您遇到不友善行为时
1. **不要直接回应**：避免争论或升级冲突
2. **记录证据**：保存相关消息或链接
3. **联系维护者**：通过邮件或私人方式报告
4. **寻求支持**：如果需要，寻求其他社区成员的支持

### 当被指出不友善行为时
1. **倾听反馈**：认真考虑他人的观点
2. **道歉（如果适当）**：真诚地道歉并解释
3. **纠正行为**：立即停止不当行为
4. **学习和改进**：从中学习并避免重复

### 维护者的角色
* **中立调解**：公平处理所有报告
* **及时响应**：尽快处理问题报告
* **保护隐私**：保护报告者的身份
* **教育指导**：帮助社区成员理解和遵守准则

## 后果

不当行为可能导致以下后果之一：

* 📝 **私下警告**：通过私人消息提醒
* 🔒 **公开警告**：在公共渠道中正式警告
* ⏸️ **临时暂停**：暂时限制社区参与
* 🚫 **永久禁止**：永久禁止参与项目
* 📋 **官方道歉**：要求公开道歉（如适用）

## 报告流程

### 如何报告问题
1. **邮件报告**：发送邮件至 yedazhi@c2h4.cn
2. **包含信息**：详细描述事件，包括时间、地点、涉及人员
3. **提供证据**：如果有截图、链接或其他证据
4. **期望结果**：说明您希望如何处理

### 报告后将发生什么
1. **确认收到**：我们会在 24 小时内确认收到报告
2. **评估情况**：维护者会调查和评估报告
3. **采取行动**：根据情况采取适当的纠正措施
4. **跟进反馈**：向报告者提供处理结果的反馈（在适当范围内）

## 持续改进

我们致力于不断改进我们的社区环境：

* **定期审查**：定期审查和更新行为准则
* **收集反馈**：积极收集社区反馈
* **培训教育**：为维护者提供相关培训
* **透明度**：在保护隐私的前提下保持透明
