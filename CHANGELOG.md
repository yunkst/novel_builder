## [1.0.0] - 2025-12-15

### 新增
- 🎉 Novel Builder 初始版本发布
- 📱 Flutter 跨平台移动应用
- 🌐 FastAPI 后端服务
- 🗄️ PostgreSQL 数据库缓存
- 🕷️ 多站点小说爬虫系统
- 🤖 AI 增强功能集成
- 🐳 Docker 容器化部署

### 核心功能
- ✅ 小说搜索功能（支持 5+ 个站点）
- ✅ 本地书架管理
- ✅ 章节缓存和离线阅读
- ✅ 阅读进度跟踪
- ✅ AI 特写功能
- ✅ 角色卡提取
- ✅ 场景插图生成
- ✅ WebSocket 实时通信

### 技术栈
- **前端**: Flutter 3.0+, Dart
- **后端**: FastAPI, Python 3.11+
- **数据库**: PostgreSQL, SQLite
- **爬虫**: BeautifulSoup4, Playwright
- **部署**: Docker, Docker Compose
- **AI**: Dify 工作流集成

### 支持的小说站点
- AliceSW (轻小说文库)
- 书库 (Shukuge)
- 小说网 (Xspsw)
- 我的书城 (Wdscw)
- 爱尚小说

## 版本说明

### 版本格式
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 发布周期
- **主版本**：根据需要发布
- **次版本**：每月发布（如有新功能）
- **修订版本**：根据需要发布（bug 修复）

### 支持政策
- **当前版本**：完全支持
- **前一个主版本**：关键安全和 bug 修复
- **更早版本**：不再维护
