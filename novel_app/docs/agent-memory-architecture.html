Novel Builder — Agent "越用越聪明" 架构

用户主动触发反思 → LLM 提取记忆 → 用户审核 → 持久化 → 下次对话自动注入

主动反思流程（8 步）

  • • 用户点击 📝 反思按钮
  • • Notifier 收集未反思消息
  • • AgentMemoryService 调用 LLM
  • • LLM 返回结构化记忆 JSON
  • • 用户审核（勾选/取消）
  • • 确认后写入 agent_memory 表
  • • 去重：相似度 > 80% 更新而非新增
  • • 下次对话自动注入 system prompt

agent_memory 表设计

  • • scenario_id: 'writing' | 'webview_extract'
  • • memory_type: preference | correction | pattern | fact
  • • key + content: 记忆键值对
  • • trust_score: 0.0-1.0 信任评分
  • • use_count: 被引用次数
  • • last_used_at: 上次引用时间
  • • UNIQUE(scenario_id, key) 防重复

与 Hermes Agent 的对比

  • • 主动触发 > 自动 Nudge（用户可控）
  • • SQLite > 文件存储（移动端友好）
  • • 用户审核 > 自动写入（透明可信）
  • • 2 场景 > 通用 Agent（精准聚焦）
  • • 无需 Curator（记忆量小）
  • • 无需 Honcho（用户画像简单）
  • • 无需轨迹压缩（不训练模型）
