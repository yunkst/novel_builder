FEATURES

为追书而生的每一处细节

从书架管理到 AI 增强，覆盖找书、读书、品书的完整体验。

📚

书架管理

本地收藏与阅读进度跟踪，多端一致的私人书库，打开即续读。

🔍

跨站统一搜索

聚合 11 个小说站点，一次搜索，横向比价章节来源与更新进度。

💾

离线缓存阅读

章节内容本地 SQLite 缓存，断网也能流畅翻页，节省流量。

🤖

AI 对话与工作流

DSL Engine 本地工作流 + Agent Chat 智能对话，信息提取与创意写作两不误。

🎨

AI 场景插图

ComfyUI 后端生成场景插图，支持负向提示词，把文字读成画面。

🕸️

角色关系图

智能提取章节角色信息，可视化人物关系网络，理清长篇人物脉络。

QUICK START

三分钟跑起来

使用 Docker Compose，一键拉起后端 API、数据库与爬虫服务。

bash 
# 1. 克隆项目
git clone https://github.com/yunkst/novel_builder.git
cd novel_builder

# 2. 配置环境变量
cp .env.example .env

# 3. 一键启动所有服务
docker-compose up -d

# 后端 API 文档 → http://localhost:3800/docs

移动应用：进入 novel_app/ 执行 flutter pub get && flutter run
